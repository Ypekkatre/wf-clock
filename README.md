
Another Warframe Clock Overlay
==============================

Overlay for Warframe on Linux that shows the remaining time at Plains of Eidolon.

![Screenshot](https://i.imgur.com/IaAFmqV.png)
![Screenshot](https://i.imgur.com/oFrsany.png)

Requirements
----

- python-requests
- python-pyqt5
- python-xlib

Usage
----

Run through console

```
python wf-clock.py
```

`Ctrl + C` to stop
