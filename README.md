
Another Warframe Clock Overlay
==============================

Overlay for Warframe that shows the remaining Plains of Eidolon time.

![Screenshot](https://i.imgur.com/IaAFmqV.png)
![Screenshot](https://i.imgur.com/oFrsany.png)

Requirements
----

1. Linux
   - python-requests
   - python-pyqt5
   - python-xlib
2. Windows
   - requirements.txt

Usage
----

Run through console

```
python wf-clock.py
```

`Ctrl + C` to stop
