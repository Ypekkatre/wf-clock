
Another Warframe Clock Overlay
==============================

Overlay for Warframe on Linux that shows the remaining time at Plains of Eidolon.

![Screenshot](https://i.imgur.com/UepvIkb.png)
![Screenshot](https://i.imgur.com/Yd5tljO.png)

Requirements
----

- python-requests
- python-pyqt5
- python-xlib

Usage
----

Run through console

```
python main.py
```
